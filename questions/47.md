question: Перечислите методы кластеризации, реализация которых в scikit-learn допускает анализ новых данных, и методы, которые нельзя применить для новых данных. Попытайтесь объяснить почему (не просто сказать, что есть predict и есть только fit_predict)
module: Практика
---

В scikit-learn **предсказать кластеры для новых данных можно там, где алгоритм даёт явные опорные точки или модель кластеров.** Например, **k-means** после обучения знает координаты центроидов кластеров - новый объект можно просто приписать к ближайшему центроиду (в sklearn у KMeans есть метод predict). **Иерархическая EM-кластеризация (GaussianMixture)** также позволяет для новой точки посчитать вероятность принадлежности к каждому из кластеров (то есть определить, к какому гауссовскому компоненту она скорее всего относится). **BIRCH** тоже сохраняет иерархию кластеров и может инкрементно добавлять новые данные. Эти алгоритмы создают **модель** (набор параметров: центры, распределения и т.д.), по которой можно классифицировать новые точки.

А вот **методы без прямой поддержки новых данных** - это такие алгоритмы, как **агломеративная кластеризация**, **спектральная кластеризация**, **Mean-Shift**, **DBSCAN**. Они не обучают явной функции или параметров для кластеров, которые можно применить потом. Например, **агломеративная кластеризация** просто строит дерево кластеров по конкретному набору точек - если появляется новая точка, встроить её без пересчёта нельзя. **DBSCAN** определяет кластеры как плотные области исходного набора; для новой точки нельзя однозначно решить, кластер она или шум, не выполнив алгоритм заново (хотя можно определить, есть ли в eps-окрестности точки достаточно объектов, но sklearn не имеет метода для этого по умолчанию). **Спектральная кластеризация** основывается на собственных векторах матрицы похожести исходных точек - добавление новой точки нарушает эту матрицу, и опять же нужен пересчёт. В итоге: методы, которые учат "модель кластеров" (центроиды, распределения), позволяют прогнозировать кластеры и для новых данных. Методы, которые просто разово кластеризуют заданный набор, для новых данных неприменимы без запуска алгоритма заново.
