Замыкание - это функция, которая ссылается к переменным вне ее тела. Функция имеет доступ к связанным переменным, а также может присваивать им значения. Пример:

```go
package main

func accum() func(int) int {
	sum := 0
	return func(x int) int {
		sum += x
		return sum
	}
}
```

Функция accum возвращает замыкание.
