Дан слайс целых чисел. Напишите функцию remove, удаляющую все нули. Примеры:

```text
remove([]) -> []
remove([0]) -> []
remove([1,0,0,2]) -> [1,2]
```

Ответ

```go
package main

func remove(in []int) []int {
	i := 0
	j := 0
	for ; i < len(in); i++ {
		if in[i] != 0 {
			in[j] = in[i]
			j++
		}
	}
	return in[:j]
}
```
